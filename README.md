# Mi Studio

Mi Studio is a premium personal portfolio for a web developer and UI/UX designer. The experience combines a polished landing page, project showcase, blog, contact flow, and an admin login for content management.

## Stack
- HTML5
- CSS3
- JavaScript
- PHP
- JSON-backed content storage
- MySQL schema included in database/database.sql

## Local setup
1. Place the project in your XAMPP htdocs folder.
2. Start Apache and MySQL in XAMPP.
3. Open http://localhost/Mi_studio/home.html.
4. Sign in to the admin dashboard at http://localhost/Mi_studio/login.php with:
   - Email: admin@mistudio.dev
   - Password: Admin@2026!

## Notes
- The current build uses JSON files for content and a SQL schema for future database-backed expansion.
- The authentication flow is prepared for admin-only access, CSRF protection, password hashing and secure sessions.
